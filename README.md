# Browsly Downloads

Official public download repository for **Browsly**.

Browsly is a visual browser automation builder for creating and running repeatable browser workflows without writing code.

## Download

Latest Windows builds will be available from the **Releases** section:

👉 Go to **Releases** → download the latest `.exe` installer.

## Current Status

Browsly is currently in early testing.  
This repository is used only for publishing downloadable builds and release notes.

## What Browsly Helps With

- Browser automation workflows
- QA and regression testing
- Data extraction workflows
- Repetitive browser tasks
- Internal dashboard/reporting automation

## Installation

1. Download the latest installer from Releases.
2. Run the `.exe` file.
3. If Windows SmartScreen appears, click **More info** → **Run anyway**.
4. Launch Browsly from the Start Menu or desktop shortcut.

## Release Notes

Each release contains:

- Installer file
- Version number
- Bug fixes
- New features
- Known issues, if any

## Important Notice

Browsly is currently provided for testing and feedback.  
Please do not use it for sensitive production workflows yet.
